# Launch Checklist

## 功能

- [ ] 首页可以进入 `works`
- [ ] `works` 可按 tag 和 city 过滤
- [ ] 每张照片可进入详情页

## 内容

- [ ] 至少 20 张首发作品
- [ ] 每张照片包含完整元数据（地点、时间、器材、标签）
- [ ] 标签命名规范一致（推荐英文 slug）

## 体验

- [ ] 桌面端无明显布局错位
- [ ] 手机端可正常浏览
- [ ] 图片加载速度可接受

## 发布

- [ ] Vercel 环境变量已配置
- [ ] 生产域名可访问
- [ ] Sanity 新增内容可在前台显示
